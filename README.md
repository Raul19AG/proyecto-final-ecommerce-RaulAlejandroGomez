# Proyecto Práctico - Sitio Web de Empresa

## Descripción

Este proyecto es un sitio web estático para una empresa ficticia, desarrollado como ejercicio práctico en el curso de Frontend de Talento Tech. El sitio incluye páginas principales como inicio, productos, reseñas y contacto, con un diseño responsivo y elementos multimedia.

## Características

- **Página de Inicio**: Incluye un video de presentación con subtítulos, sección "Nosotros" y navegación principal.
- **Página de Productos**: Muestra productos destacados con imágenes, descripciones y precios.
- **Página de Reseñas**: Contiene reseñas de clientes en un diseño de cuadrícula.
- **Página de Contacto**: Formulario de contacto y tabla de mensajes/contactos.
- **Diseño Responsivo**: Utiliza CSS Flexbox y Grid para layouts adaptables.
- **Multimedia**: Videos con subtítulos y enlaces a redes sociales.

## Tecnologías Utilizadas

- **HTML5**: Estructura del sitio web.
- **CSS3**: Estilos y diseño, incluyendo Flexbox y Grid.
- **Google Fonts**: Fuentes personalizadas (Bitcount Grid Double).
- **Material Symbols**: Iconos de Google.
- **Formspree**: Para el envío de formularios de contacto.

## Estructura del Proyecto

```
practicos/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── assets/             # Recursos multimedia
│   ├── subtitulos.vtt  # Subtítulos para video
│   └── ico/            # Iconos y logos
├── pages/              # Páginas adicionales
│   ├── contacto.html   # Página de contacto
│   ├── productos.html  # Página de productos
│   └── reseñas.html    # Página de reseñas
└── README.md           # Este archivo
```

## Cómo Ejecutar el Proyecto

1. Clona o descarga el repositorio.
2. Navega a la carpeta `practicos/`.
3. Abre el archivo `index.html` en tu navegador web preferido.

No se requiere servidor web para visualizar el sitio, ya que es completamente estático.

## Notas

- Las imágenes de productos se cargan desde Picsum Photos (servicio de imágenes aleatorias).
- El formulario de contacto utiliza Formspree para el envío de datos.
- Algunos enlaces a redes sociales están incluidos en el footer.
